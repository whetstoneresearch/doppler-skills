# V2 Migrator Verification

## Checklist

1. Verify Airlock configuration points `liquidityMigrator` to the intended V2 migrator.
2. Verify migration transaction includes V2 destination interactions.
3. Verify destination liquidity was minted/updated for the expected pair.
4. Verify recipient ownership and post-migration balances.
